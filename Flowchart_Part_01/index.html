Common flowchart symbols:
This symbol marks the beginning or ending of the process.
Process
This is the most common symbol. It is used for most process steps.
Decision Making
This symbol represents a decision in the flow chart. Usually an arrow that flows from this symbol is noted as a “Yes” or “No” decision.
Document
This symbol represents a document created or prepared as part of the process (e.g., a report).
Storage Device
This symbol represents a physical storage device for data such as a hard drive. The focus of this symbol is the media, (i.e., “where to store.”)
